Program that calculates: 
1) Netmask 
2) Wildcard
3) Network address
4) Broadcast address
5) First Host Internet Protocol (FHIP) Address 
6) Last Host Internet Protocol (LHIP) Address 
7) Number of Host Bits 
8) Number of host addresses
8) Number of Borrowed Bits  
9) Number of subnets
10) Subnet Index
